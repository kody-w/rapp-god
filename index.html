rapp-god

Every part of the RAPP ecosystem lives in more than one repo. This page reads manifest.json and, right now in your browser, fetches each part's raw file from every repo that carries it and hashes them live — so you can see, at a glance, whether the copies still agree, and exactly what differs when they don't. No server, no build in the path — just raw.githubusercontent.com and a SHA‑256. It observes; it never fixes.

The map — which repo houses which part

How it works. The list of parts and their source URLs is in manifest.json (the only hand-authored file). For each part this page fetch()es every source straight from raw.githubusercontent.com and hashes it with crypto.subtle — the comparison is computed here, live, every time you load. Identical hash = in sync; different = a fork (hit "diff" to see what changed). Optionally it also reads registry.json to show every past version held as a pinnable fallback — but the live verdict doesn't depend on it.
