RAPP Brainstem for VS Code

Pages instead of source. Hatched twins as tabs. The brainstem UI inside your editor. Built for non-technical RAPP users.

⬇ Download extension .vsix View source on GitHub

Install in 10 seconds

  1. Click Download extension above.
  2. Open VS Code. Click the Extensions icon in the activity bar (or press Cmd+Shift+X).
  3. Drag the downloaded .vsix file straight into the Extensions panel. Done.

Prefer the terminal? code --install-extension ~/Downloads/rapp-brainstem.vsix

What lands in VS Code

Features

  • HTML → rendered page Open any .html file and see the page, not the source. One click flips back to source.
  • Markdown → styled doc Same for .md. Theme-aware. Live updates as you type.
  • Twins dashboard Every hatched twin shows up in the sidebar with live running status. Click to open in a tab.
  • Right-click → Hatch Right-click any folder in the Explorer to spin up a new project twin via the global brainstem.
  • Embedded brainstem UI The live brainstem on localhost:7071 docked in your VS Code sidebar.
  • OS-aware boot No brainstem yet? One click runs the official installer — bash on macOS/Linux, PowerShell on Windows.

What you need

  • VS Code 1.85+
  • The RAPP brainstem installed locally (the extension will offer to install it for you on first boot)
