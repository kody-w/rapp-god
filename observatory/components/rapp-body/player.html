    Serve this directory to watch the biography

    The player loads frames/index.json and each frame with a relative fetch. Browsers block file:// fetches, so open it over a tiny static server instead:

    cd rapp-body && python3 -m http.server 8000

    then visit http://localhost:8000/player.html. On GitHub Pages it just works.
