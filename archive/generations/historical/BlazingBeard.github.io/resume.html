The real superpower: new agent in minutes

Counting agents misses the point. The leverage is the factory: any new human task that walks through the door becomes a working teammate before the meeting ends. Three paths in, one swarm out.

① Generate
From a sentence. Describe the job — "watch this folder, summarize new PDFs, post to Teams" — and a new agent is scaffolded, registered, and live on the next request. No deploy, no PR, no ticket.
② Assimilate
From the wild. Point at any GitHub repo, API, CLI, or doc. Borg ingests it, distills the useful surface, and wraps it as a single-file agent the LLM can invoke like any other tool. Other people's work becomes your team's reflex.
③ Modify
Fork an existing one. Hot-reload from disk means edit-save-use. A 30-line change turns a generic researcher into a domain specialist — pricing analyst, compliance checker, model reviewer — without restarting anything.

Translation for the hiring manager: the question changes from "do you have an agent for X?" to "how soon do you need it?" Answer is almost always this afternoon.
