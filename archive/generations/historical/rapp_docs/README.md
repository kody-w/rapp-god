# 📚 RAPP Engineering Docs

**Every RAPP spec in one place — streamed live from each home repo's GitHub raw data. No copies, no server.**

Live: **https://kody-w.github.io/rapp_docs/**

This is a single static page. `specs.json` is the index; each entry points at a spec's `raw.githubusercontent.com` URL in whatever public repo it lives in. The page fetches and renders it **live**, so the docs are always current with each repo's `main` (or branch) — nothing is duplicated here. Add a spec by adding one line to `specs.json`.

Engine, not experience. Static, raw-driven, forkable.
