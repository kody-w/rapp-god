How to Use

    About this Mashup

    This app pairs the W3C Verifiable Credentials Data Model 2.1 with the same ECDSA P-256 keys that secure your ZooCoin wallet. Both standards use the same elliptic curve, so one keypair can sign blockchain transactions and issue tamper-evident JSON-LD credentials. Verification runs entirely in your browser using the Web Crypto API.
