How to Use

    About this Mashup

    Storage Buckets is not yet a /TR/ Recommendation, so this app uses W3C IndexedDB 3.0 as the underlying store. When the experimental navigator.storageBuckets API is detected, the IndexedDB connection is opened inside an isolated bucket; otherwise the default bucket is used. Either way, UTXO records are stored in an object store with a composite key on [txHash, outputIndex] and a secondary index on address, enabling O(log n) balance lookups instead of the O(n) JSON scan that localStorage requires.
