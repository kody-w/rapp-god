    About this Mashup
    
    
    This miner uses W3C WebGPU compute shaders to brute-force nonces in parallel. The WGSL kernel computes a tiny "lite-hash" digest derived from the block header bytes; hashes with the required leading-zero prefix are flagged as valid. The result is then re-validated with the canonical Web Crypto SHA-256 implementation before the block is appended to the local ZooCoin chain. When WebGPU is unavailable, mining falls back to a CPU loop using the Web Crypto API.
