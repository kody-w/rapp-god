How to Use

    About this Mashup

    The did:zoocoin method derives a DID from a ZooCoin address. Resolution looks up the DID Document in the local registry. Each DID Document follows the W3C DID 1.1 data model with @context, id, verificationMethod, and authentication sections. Key rotation works because every wallet keypair is also a verificationMethod — the chain of trust runs from the on-chain identity through the document history.
