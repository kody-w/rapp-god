How to Use

    About this Mashup

    WebAuthn Level 3 produces ES256 signatures on a hardware authenticator (Touch ID, Windows Hello, FIDO2 key, passkey). Your existing ZooCoin ECDSA P-256 software key signs the same transaction body in parallel. A transaction is considered dual-proofed only if both signatures verify. The hardware key never leaves the secure element; the software wallet provides the on-chain identity.
