SHADER // SYNTH — REFERENCE

VISUAL DSL

Chain operators, end with .out(). Each chain compiles to GLSL.

SOURCES

  • osc(freq, sync, offset) — oscillating bars
  • noise(scale, offset) — perlin-ish noise
  • voronoi(scale, speed, blending) — cell pattern
  • shape(sides, radius, smoothing) — polygon
  • gradient(speed) — rainbow gradient
  • solid(r, g, b) — flat color
  • src(o0) — feedback from previous frame

GEOM

  • .rotate(angle, speed)
  • .scale(amount, xMult, yMult)
  • .repeat(rx, ry)
  • .kaleid(n)
  • .pixelate(x, y)
  • .modulate(other, amount)
  • .modulateRotate(other, mult, offset)
  • .modulateScale(other, mult)

COLOR

  • .color(r, g, b)
  • .brightness(n) · .contrast(n)
  • .invert() · .luma(threshold)
  • .posterize(n, gamma)
  • .saturate(n)

BLEND

  • .blend(other, amount)
  • .add(other, amount) · .mult(other, amount)
  • .diff(other) · .layer(other)

AUDIO DSL

Strudel-flavored mini patterns. Whitespace = steps, ~ = rest, *N = repeat.

SOURCES

  • s("bd sn hh cp") — sample names
  • n("c2 e2 g2") — note names or midi numbers

SAMPLES

  • bd — kick
  • sn — snare
  • hh — hat
  • cp — clap
  • bass / lead / pad — synths (use with n())
  • sine / square / saw / tri — basic synth waves

MODIFIERS

  • .gain(n) · .pan(n)
  • .lpf(hz) · .hpf(hz)
  • .delay(time) · .room(amt)
  • .speed(n) · .cycles(n)

MIDI

Use midi(cc, lo, hi, min, max) in any visual or audio expression.

  • osc(midi(1, 0, 127, 10, 80)).out()
  • MIDI notes trigger built-in samples by pitch class.

KEYS

  • Cmd/Ctrl + Enter — evaluate current block
  • Cmd/Ctrl + Shift + Enter — evaluate full editor
  • Cmd/Ctrl + . — stop audio
  • ? — toggle this help
