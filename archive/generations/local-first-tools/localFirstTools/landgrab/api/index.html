The Zero-Server API

No backend. No auth. No cost. Just predictable static URLs on raw.githubusercontent that any app, script, or AI agent can fetch(). This is planetary-scale read API infrastructure for $0 — see the HQ.

Recipes

# every tool as JSON
curl -s https://kody-w.github.io/localFirstTools/landgrab/index.json | jq '.count, .categories'

# search client-side
curl -s .../landgrab/index.json | jq '.apps[] | select(.tags|index("3d")) | .url'

# fetch a tool's full source
curl -s https://raw.githubusercontent.com/kody-w/localFirstTools/main/apps/games/snake4.html
