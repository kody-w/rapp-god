Available Now

Prefer the terminal? Use Ollama for CLI access to the same models.

ollama pull gemma2:2b && ollama run gemma2:2b

Quick Start

Three steps. Under a minute.

1

Open the Playground

Click the button below or navigate to playground.html. That's the entire runtime — one HTML file.

2

Pull a Model

Type pull gemma4 to download Gemma (~1.5 GB). The model caches in your browser — you only download once.

3

Start Chatting

Type run and talk to your local AI. Everything runs on-device — nothing leaves your machine.
