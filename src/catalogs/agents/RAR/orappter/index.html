AI that runs on your device

ORAPPTER uses WebGPU to run open models entirely in your browser tab. Your prompts never leave your machine. The model weights cache locally — once downloaded, it works offline forever.

  • Zero install — no download, no terminal, no Python
  • 100% private — your data never leaves your browser
  • Works offline — model cached locally after first pull
  • No rate limits — run as many prompts as you want, forever free
Try it now →

Programmable with LisPy

ORAPPTER isn't just a chat box. It's a full runtime with a built-in Lisp interpreter. Write agents, chain models, build pipelines — all in one browser tab.

  • Full LisPy REPL — evaluate expressions alongside chat
  • Agent cards — activate any card as an AI persona
  • Knowledge base — store and query with local embeddings
  • Same API everywhere — agents run in browser or CLI

orappter> (+ 1 2 3)

6

 

orappter> (activate-card "@borg/card_smith_agent")

✓ Activated CardSmith as AI agent.

ORAPPTER vs Ollama

Same idea — run AI locally. Different approach. ORAPPTER runs in your browser. No install. No terminal. Open a tab and go.

Ollama is great if you want CLI tooling and server-mode. ORAPPTER is for everyone else.
