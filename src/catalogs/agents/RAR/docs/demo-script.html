🎬 RAPP Engineering Demo Script

5–10 minute live demo for Microsoft engineering leadership  ·  Three pillars, one story

Run of Show

Total~8 minutes (tight) to ~12 minutes (with Q&A buffer)
Pillar 1Agent Conversion — n8n workflow → RAPP → Copilot Studio (~3 min)
Pillar 2Agent Creation — Live transcript → pipeline → deployed prototype (~3 min)
Pillar 3The Platform — Patent, hot-loading, registry, cards, open-source (~2–3 min)
ClosePartnership ask + call to action (~1 min)

Opening — Set the Stage

SAY

"Thanks for having me. I'm going to show you three things in the next eight minutes — and I'm not going to tell you what the platform can do. I'm going to show you what we use today, in production, for real customer engagements."

SAY

"Three pillars. First: agent conversion — take something built outside our ecosystem and bring it into Copilot Studio. Second: agent creation — go from a customer conversation to a working prototype. Third: the platform — the patented engine that makes it all work."

Pillar 1 — Agent Conversion

~3 MINUTES   The headline: take a competitor workflow and bring it home to Copilot Studio.

Setup

SAY

"Let's start with the thing everyone's been asking about. Can we really convert agents from other platforms into Copilot Studio? Let's find out."

SHOW

Open n8n in a browser tab. Display a non-trivial workflow — email intake, CRM lookup, conditional routing, response generation. Something with 8–12 nodes that clearly does real work.

SAY

"This is an n8n workflow — a direct competitor to Power Automate. It handles inbound email, looks up the customer in CRM, routes based on priority, and drafts a response. It's real. It works. And it's not in our ecosystem."

The Conversion

DO

Export the n8n workflow JSON. Feed it into the RAPP transpiler agent. Show the output: a clean RAPP agent .py file with a __manifest__ and perform() method.

SAY

"One command. The transpiler reads the n8n JSON, understands the nodes, the connections, the logic — and produces a single-file RAPP agent. Same behavior. Our platform."

DO

Now run the Copilot Studio transpiler on the RAPP agent. Show the MCS-compatible output.

SAY

"And now one more step — RAPP to Copilot Studio. Same agent. Same logic. Now it's a Copilot skill that power users can extend without writing code."

Side-by-Side

SHOW

Split screen: n8n on the left, Copilot Studio on the right. Trigger the same input on both. Show identical outputs.

SAY

"Same input. Same output. n8n on the left. Copilot Studio on the right. That's agent conversion."

TRANSITION

"Now — if I can convert code-based agents, Anthropic agents, n8n workflows — that's powerful. But what about creating agents from scratch? That's where it gets really interesting."

Pillar 2 — Agent Creation

~3 MINUTES   Transcript to working prototype. The production pipeline.

The Transcript

SHOW

Flash the raw transcript on screen — a real customer discovery call. Highlight key details: customer name, pain points, desired outcomes.

SAY

"This is a real discovery call transcript from an account team engagement. The customer described what they need. Now watch what happens when we feed this into the RAPP pipeline."

The Pipeline

DO

Run the RAPP pipeline. Show it processing: transcript analysis → agent generation → validation → deployment. Either live or via the pre-recorded pipeline video with narration.

SAY

"The pipeline reads the transcript, extracts the use case, generates the agent code, validates the manifest, runs contract tests, and deploys it. What used to take weeks takes minutes."

The Result

SHOW

Open Copilot Studio. Show the deployed agent responding to a real query. The agent works.

SAY

"From a conversation to a working Copilot Studio prototype. The account team can show this to the customer tomorrow. That's what 20x faster looks like."

TRANSITION

"So we can convert agents in. We can create agents from scratch. But what's underneath all of this?"

Pillar 3 — The Platform

~2–3 MINUTES   The patented engine. Hot-loading. Registry. Open-source. The full picture.

The Patent

SAY

"Everything you just saw runs on a platform we built called RAPP — the Rapid Agent Prototype Pattern. We filed the patent on this before anyone else was talking about agent ecosystems. Hot-loading agents. Universal memory across channels. The waiter pattern for orchestration. All of it."

The Registry

SHOW

Open kody-w.github.io/RAR. Browse the registry — 138 agents, 19 categories. Click a publisher filter. Open an agent detail. Show the deep link.

SAY

"138 agents. 7 publishers. 19 industry categories. B2B sales, healthcare, financial services, government — every vertical has templates ready to deploy. This is the open-source agent registry. Anyone can browse it, install agents, or contribute."

Hot-Loading

SAY

"Here's what makes the architecture special. Agents live in Azure File Storage. Drop a new .py file — it's live instantly. No redeployment. No build pipeline. No downtime. We had this working before anyone else announced agent hot-loading."

The Cards (Quick Hit)

SHOW

Open the Agent Store. Switch to Cards view. Show a holographic card. Open a pack if time allows.

SAY

"And because we believe agents should be fun to discover — every agent gets a trading card. Deterministic. Offline. Self-assembles from 18 characters. It's the discoverability layer that makes people actually want to browse the registry."

Open Source

SAY

"The entire thing is open source. The registry is a GitHub template — any org can fork it, run their own internal registry, and optionally federate back upstream. We built it to be the npm of AI agents."

Close — The Ask

~1 MIN Partnership + call to action

SAY

"So here's the picture. We can convert agents from any platform into Copilot Studio. We can create agents from customer conversations in minutes. And we have the platform — patented, open-source, with 138 agents already in the registry."

SAY

"What we want to partner on: the conversion path has some Copilot Studio limitations we've documented. We'd love to work with the engineering team to close those gaps. Because when we do — every agent, from any platform, becomes a Copilot skill. That's the opportunity."

SAY

"I'll leave you with this: everything I just showed you is live, right now, at the links I'm sharing. The installer, the registry, the architecture docs. Thank you."
