Frequently Asked Questions

Do I really not need to know how to code?

Really. The Agent Workbench provides templates. AI assistants (Claude, ChatGPT) can write the code for you. You provide the idea and the description. The machines handle the implementation. The book walks you through every step.

What's the difference between this and ChatGPT?

ChatGPT is a general-purpose AI you talk to. RAPP agents are specialized tools that do specific things. Think of it like the difference between asking a friend for directions vs. using Google Maps. Both work, but one is built for the job.

Is this another AI hype product?

The Agent Store has 131 working agents. The code is open source. The wiki has 400+ articles. The ecosystem runs on GitHub — the same platform developers trust with their code. There's no hype, just tools that work.

Why is everything free except the book?

Because the ecosystem grows when more people use it. Every new agent published, every new user who builds something — that makes the whole thing more valuable for everyone. The book is $9.99 because writing 20 chapters takes time, and Amazon needs to get paid. Everything else is MIT open source.

Can I use this at my company?

Yes. RAPP supports federation — fork the repo and run your own private agent store. Your agents stay internal. You can still pull public agents from the main store. Chapter 15 covers this in detail.

How long does it take to build an agent?

Your first agent takes about 15 minutes with the Workbench. The meeting summary agent in Chapter 10 takes about 10 minutes. Once you've built one, subsequent agents take 5-10 minutes each.
