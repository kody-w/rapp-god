Standalone agents

Live enterprise sandbox

Fourteen globally public, schema-true simulators of the systems these agents integrate with — same API shapes, field names, and id formats as the real thing, no auth, no rate limits. All fourteen share one fictional world (Aster Lane Office Systems: 22 customer accounts with cross-linked stories), so agents can join records across systems — CRM case CAS-2601xx to ITSM INC00100xx to ERP PO-470xx to HRIS TOR-10xx. Status badges below are fetched live from each system right now.

Run it yourself

The templates layer onto the Microsoft AI stack one tier at a time — the same ladder the RAPP installer teaches. Start local, add the cloud, then publish to your whole organization.

Tier 1 — The Brainstem

Local-first agent server

Run agents on your machine with GitHub Copilot as the engine — no API keys, no cloud setup. One command installs the brainstem; drop any *_agent.py from this library into its agents folder and it self-registers.

Tier 2 — The Spinal Cord

Azure Functions backend

Give the agents a cloud body: an always-on Azure Function with Azure OpenAI, persistent memory, and Entra ID auth. One click deploys the full infrastructure from this repository's ARM template.

Tier 3 — The Nervous System

Copilot Studio & Teams
