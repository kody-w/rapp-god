# RAPP Brainstem

> The Brainstem is the Tier-1 engine of RAPP: a single-file Flask server that turns a GitHub Copilot token into a local, tool-using AI agent runtime — no API keys, no cloud.

Auto-discovered `*_agent.py` agents, a `soul.md` identity, and local memory. The canonical install is the [RAPP Installer](https://github.com/kody-w/rapp-installer).
