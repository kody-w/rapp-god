Top Questions, One Slide
🧠

Four questions.
One slide of answers.

1The Copilot question

Does this compete with Copilot Studio?

No current promotion path is shipped. RAPP is experimental local source with a pre-acceptance RAPP/1 façade; Azure and Copilot Studio are retired design targets, not operational tiers.

2The offline question

Does it really run offline?

The retained local source can be evaluated offline, but there is no supported offline install or browser product. Model and dependency behavior must be verified by the operator.

3The production question

How does it get to production?

It does not currently get to production. Tier 2 and Tier 3 are retired; any future adapter needs a separate acceptance and release. A reviewed single file is source evidence, not a deployment.

4The one-sentence

What is this, in one sentence?

RAPP is experimental local agent source behind a pre-acceptance RAPP/1 boundary. It is not a shipped cloud, browser, or install product.
