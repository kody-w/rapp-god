// agent.py beats skills.md

One file = one agent. It's code, not prose. Anthropic and OpenAI ship skills.md (markdown the LLM interprets — non-deterministic). rappid ships agent.py (named functions the LLM dispatches — same input → same output, every model, every time). If you can't read the file and predict what will happen, you can't ship it.
