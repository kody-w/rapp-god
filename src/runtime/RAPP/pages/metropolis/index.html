This is a Kazaa/torrent-style index. Each entry points at a real GitHub repo (the seed). When a brainstem joins a neighborhood, that brainstem becomes a seeder. The metropolis itself is just a directory — there is no central server, no platform-operated database, no admission ceremony. Anyone can fork this index and run their own tracker. The protocol is the network, not this URL.

Schema: rapp-metropolis-index/1.0 · Tracker: kody-w · Source: index.json · protocol README

To register your own neighborhood: open a PR adding an entry to pages/metropolis/index.json, OR run your own tracker by forking this directory.
