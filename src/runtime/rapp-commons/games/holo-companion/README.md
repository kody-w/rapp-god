# Holographic Companion

A 3D companion that follows you through the Nexus; companions housed in-repo.

A **portal world** in the Commons: open it through [`portal.html`](../../portal.html). The hub loads the app (https://kody-w.github.io/localFirstTools/chrome-extension-build/apps/games/nexus-holographic-companion.html) in a sandboxed iframe and feeds it the raw-CDN URL of this folder's `state/`, so the world streams its data straight from the public commons repo. Add a scene/companion = append a signed file under `state/` and PR it.
