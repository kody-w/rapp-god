# 🔮 rapp-holo — AI interaction past chat and voice

**[SPEC.md](./SPEC.md)** defines `rapp-holo/1.0`: the model projects
interface (`|||HOLO|||` payloads) onto whatever surface exists — a gesture
pad, AR, a watch, or headless agent-to-agent. VOICE says; HOLO shows.

Reference surface: the RAPP VUI (Virtual User Interface) — a MediaPipe
gaze/pinch macro-pad flying on the
[RAPP release train](https://kody-w.github.io/rapp-train/).
