Paste this into your terminal. Bypasses the macOS "unidentified developer" warning and Windows SmartScreen entirely — works because curl-downloaded files don't get the quarantine flag the way browser-downloads do.

macOS or Linux
curl -fsSL https://kody-w.github.io/ez-rapp/install.sh | bash
Windows (PowerShell)
irm https://kody-w.github.io/ez-rapp/install.ps1 | iex

Want just the brainstem (no Electron window)? See the rapp-installer one-liner. ez-rapp and rapp-installer write to the same ~/.brainstem/, so you can use both.

How it works

  1. 1
    Download the ez-rapp app for your computer (above).
  2. 2
    Open it. First launch sets up Python and the AI brainstem on your machine. Takes about a minute.
  3. 3
    Sign in with GitHub. One click. Uses your GitHub Copilot subscription — no separate API key.
  4. 4
    Chat. Ask anything. It runs locally on your laptop.
