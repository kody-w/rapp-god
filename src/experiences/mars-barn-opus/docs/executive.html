The Opportunity

We've built an open-source Mars colony simulation that runs on real NASA weather data, real physics, and real engineering constraints. It's playable in a browser with zero installation. It teaches autonomous systems, remote operations, and critical decision-making through gameplay — not lectures.

The core insight: The same software patterns that keep a simulated Mars colony alive are the same patterns that run autonomous fleets, manage distributed infrastructure, and operate remote systems. The simulation is the training ground. The skills transfer directly.

Dependencies

🔬 Physics Engine

Mars atmosphere, solar irradiance, thermal dynamics, radiation — all NASA-sourced. Stefan-Boltzmann, Beer-Lambert, barometric formula. If the physics says the colony dies, it dies.

🤖 Autonomous AI

10 governor archetypes + LisPy VM (sandboxed Lisp) for custom AI. Colony Risk Index computed every sol. Reflex arcs fire between heartbeats. The system reacts in 50ms even though it thinks every 2 seconds.

🎮 Oregon Trail Gameplay

Mission selection → crew configuration → 3D colony view → Papers Please decision tasks → manual override minigames with light delay → post-mortem analysis. Playable in any browser.

🧠 Nervous System Architecture

Echo frames (structured deltas), inertia signals (derivatives), reflex arcs (standing orders), patrol agents (continuous responsiveness). A discrete system with continuous behavior.

Why This Matters Beyond Mars

1. Autonomous Systems Training

Every organization running autonomous systems — drones, robots, vehicles, infrastructure — faces the same problem: the system needs to survive when the operator can't respond fast enough. Our nervous system pattern (echo → reflex → patrol) is a general solution. The Mars sim teaches it through play.

2. Remote Operations

The manual override minigames train real remote ops skills: send commands with latency, diagnose from telemetry, sequence procedures correctly. "Robot can't unscrew a bolt" is the Mars version of "drone needs manual intervention in a cell tower inspection." Same skill, different domain.

3. Decision-Making Under Uncertainty

The Papers Please task system presents real tradeoff decisions with cascading consequences. Reroute O₂ and starve the greenhouse. Deny medical care and the whole crew demoralizes. Every "mundane" decision has downstream effects. This is what operations management actually looks like — the sim just makes it visceral.

4. Educational Platform

Physics, engineering, computer science (write your own AI in LisPy), robotics, data science, game theory, systems thinking, ethics. Every STEM discipline in one simulation. Zero setup. Works on any school computer. Already has 299 automated tests validating every behavior.
