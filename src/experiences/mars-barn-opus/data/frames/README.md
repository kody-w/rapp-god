# Sim Frames

Public ledger of Mars conditions. Each frame is one sol of environmental data
generated by the Rappter engine from real NASA Mars climate models.

Every player faces the **same Mars**. Your colony management determines survival.

## Format

- `manifest.json` — Index of all available frames
- `latest.json` — Pointer to the newest frame
- `sol-NNNN.json` — Individual sol frames

## Frame Schema

See `sol-0001.json` for the canonical example. Key fields:

| Field | Description |
|-------|-------------|
| `sol` | Sol number (Mars day) |
| `utc` | Earth timestamp when frame was generated |
| `mars` | Mars conditions (temp, pressure, solar, dust, wind) |
| `events` | Environmental events (storms, devils, flares) |
| `hazards` | Equipment/terrain hazards with probabilities |
| `comms` | Earth-Mars communication window |
| `terrain` | Surface conditions |
| `challenge` | Optional forced task for this sol |
| `frame_echo` | Context from previous frame for continuity |

## Competitive Play

All players consume the same frames. The leaderboard at `docs/simhub.html`
tracks who survives longest with the best score.
