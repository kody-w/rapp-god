# rapp-quests

The **experience layer** for the [RAPP Brainstem](https://github.com/kody-w/rapp-installer):
guided quests, demo scripts, and starter content. The grail repo is
"engine, not experience" by constitution — this is where the experience lives,
on its own release cadence.

## Quests

| # | Quest | You'll have at the end |
|---|---|---|
| 01 | [First Boot](quests/01-first-boot.md) | A running local brainstem you've chatted with |
| 02 | [First Agent](quests/02-first-agent.md) | A custom agent you wrote, hot-loaded without a restart |
| 03 | [Memory](quests/03-memory.md) | A brainstem that remembers you across conversations |

## Demo scripts

Battle-tested walkthroughs for showing the brainstem to someone else —
see [demos/](demos/). Each lists the pre-flight checks (run
[rapp-postflight](https://github.com/kody-w/rapp-postflight) first!), the
beats, and the recovery move if something goes sideways live.
