🛫 RAPP Flight Deck

Pre-release rings of the RAPP Brainstem, runnable on any machine straight from this page — no VPN, no LAN, no signup. Freshest first: Canary → Nightly → Alpha → Beta → the stable Grail release.

⚠️ These are test builds. They can break — that is their job. Each ring is qualified by CI and an attestation chain before promotion, but only the Grail is the supported product. Test flights are fully sandboxed (~/.rapp-flight/, port 7075) and never touch an existing install. Joining a ring makes it this machine's real install — switch rings by wiping ~/.brainstem and joining another.

Test-fly a ring (sandboxed — safe next to any existing install)

macOS / Linux / WSL — pick your ring by changing the last word
curl -fsSL https://kody-w.github.io/rapp-canary/flight.sh | bash -s -- canary
curl -fsSL https://kody-w.github.io/rapp-canary/flight.sh | bash -s -- beta

Fly a specific feature (a "flight")

Any branch on rapp-canary is a flyable feature. Sandbox-run it with the branch name as the second argument:

curl -fsSL https://kody-w.github.io/rapp-canary/flight.sh | bash -s -- canary <branch-name>
