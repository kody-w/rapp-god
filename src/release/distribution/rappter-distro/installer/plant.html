What does this do?

Creates a public GitHub repo at <your-handle>.github.io/<your-name>, drops in the byte-identical RAPP kernel, generates your front-door page, and enables GitHub Pages. The repo and the URL are yours; nothing depends on a central server. Mirror Spec · The Front Door
