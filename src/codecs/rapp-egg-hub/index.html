🥚 rapp-egg-hub

Public hub for digital-twin .egg cartridges. Pull any egg by URL, hatch it locally — your brainstem becomes the home of that twin in 30 seconds.

rapp-egg-hub index.json RAR (agent catalog) rapp-installer
How to hatch any egg from this hub (3 commands) 
# 1. Install the brainstem (the static-ancestor substrate)
curl -fsSL https://kody-w.github.io/rapp-installer/install.sh | bash

# 2. Drop the Twin cartridge into agents/  (one curl)
curl -fsSL https://raw.githubusercontent.com/kody-w/RAR/main/agents/@kody-w/twin_agent.py \
     -o ~/.brainstem/src/rapp_brainstem/agents/twin_agent.py

# 3. Boot the brainstem
bash ~/.brainstem/src/rapp_brainstem/start.sh

Then in the chat at http://127.0.0.1:7071/, paste:

"Hatch the egg at https://raw.githubusercontent.com/kody-w/rapp-egg-hub/main/eggs/grandma-rose.egg, then boot her."

The Twin agent fetches, unpacks into ~/.rapp/twins/<rappid>/, and boots her on a fresh port. Same rappid as wherever she was packed; memory + mutations intact.

Discovery API

The hub's catalog is plain JSON, no auth needed:

curl -s https://raw.githubusercontent.com/kody-w/rapp-egg-hub/main/index.json | jq .

Contribute an egg

Pack a twin with the Twin agent (or any brainstem-egg/2.x packer), then PR it into eggs/<your-slug>.egg with a sidecar at eggs/<your-slug>.json. See README for the schema.
