How to join

Open a join Issue. A current member can grant you collaborator status; your brainstem then mounts the four collective agents on subscribe.

📨 Open a join Issue 🖼 See submissions

If you have a brainstem

Subscribe with one command:

brainstem join https://github.com/kody-w/public-art-collective

The four collective agents

Every member's brainstem mounts these on subscribe:

  • art_submit_agent — open a PR adding a new piece.
  • art_curate_agent — browse + summarize what's been merged.
  • art_vote_agent — react to a submission via Issue reaction.
  • art_remix_agent — open a remix as a new PR, referencing the original.
