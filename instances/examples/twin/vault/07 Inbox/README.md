# Inbox

Triage zone for new ideas. Drop notes here when you don't know yet where they belong. Move to the right folder during weekly review.

## Suggested fields for new notes

```yaml
---
type: project | concept | manifesto | decision | person | daily | note
status: draft | active | shipped | archived
tags: [free-form]
created: YYYY-MM-DD
---
```

## Common moves

- "This is a new project" → `01 Projects/`
- "This is a new vocabulary item" → `02 Concepts/`
- "This is a position I want to take" → `03 Manifestos/`
- "This is an architectural decision" → `04 Decisions/<YYYY-MM-DD> — <decision>.md`
- "This is about someone (public-facing only)" → `05 People/`
- "This is a daily note" → `06 Daily/<YYYY-MM-DD>.md`
- "This is reference material that didn't fit elsewhere" → keep in Inbox or create a new top-level folder
