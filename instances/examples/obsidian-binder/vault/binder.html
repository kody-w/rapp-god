Summon a card into your vault

Paste a 7-word incantation. The federation walker resolves it via peers.json, fetches the card payload, and renders the markdown note you should save into vault/cards/.

Browser security note: this page can fetch the federation files (they're on raw.githubusercontent.com) and show you the markdown to copy. It cannot write directly into your vault — copy the output into a new file under vault/cards/ and commit. This keeps the vault as the source of truth.
How this works

1. Fetch peers.json at the repo root.

2. For each peer, fetch their seed_index.

3. Compute the seed from the incantation (locally, via the canonical 1024-word mnemonic).

4. Walk the indexes for the matching seed → fetch the card JSON.

5. Render the JSON as a vault-ready markdown note. You paste it into a new file.

The seed → words conversion uses the same wordlist as every other binder; see SPEC §4.
