# Core 2 design draft

This directory is an inert, non-normative design model. It is not imported,
executed, or advertised as a compatibility profile. The implemented runtime
remains `lispy-core@1`.

`model.json` records unresolved Form/Vector and worker-selection decisions so
future conformance work has a reviewable starting point without changing the
current reader, evaluator, wire format, or worker protocol.
