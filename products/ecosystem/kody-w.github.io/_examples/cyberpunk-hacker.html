What this is

A narrow neon-lit Tokyo alley at night. 22 GLSL puddles with animated ripple + neon swirl shaders. 1800 instanced rain streaks. Tall buildings on either side with procedural canvas-drawn kanji + Latin neon signs bound to colored point lights. Floating holographic billboards. Color palette discipline: pink + cyan + yellow against deep blue/black. WASD + mouse + pointer lock. TAB toggles CSS-overlay HUD: corner bars for HP/B-time/deck, ammo, target ID with ray-vs-sphere check on guards in frustum. Q activates brief bullet-time (camera slows, glowing bullet trails, chromatic aberration kicks in). E to start a 3-glyph hack mini-game on the secure door. Inside: 3-4 corp guards with peek-fire AI in cover. 110 BPM synthwave bass + kick + arpeggiated lead via Web Audio.

Why this is mind-blowing

Cyberpunk 2077 had a $300M budget and a buggy launch. This nails the surface-level identity in 1645 lines from one paragraph — the rain, the puddles, the neon, the holographic HUD, the bullet-time, the hack mini-game. Color discipline is half the work.
