Sixty ambient penguins waddle outside. Dr. Amaru Quist (ice core lead — pulled core to 247m), Dr. Vana Trill (microbiologist — slide-mounting samples), Engineer Bren Halloway (generator maintenance — gauge dance), Cook Pip Crowsbridge (cocoa + soup — bottomless pot), Comms Tess Felling (radio static + family check-in calls), Pilot Jules Mendoza (bush pilot — landed yesterday with mail). Each researcher has a goal, a need, a plan, a current step, a mood, AND a Cabin-Fever meter — all visible in the AGENTIC THINK PANEL. Plus Captain Crunch the husky with his own brain ("greet → belly rub → patrol mess hall → nap by generator → repeat") — when his brain panel is focused, the icon swaps to 🐺. Plus 30 named penguins outside (Stumpy, Waddleton, Tuxedo, Dr. Floof). Weather overlay top-right shows windchill -47°C / aurora 73% / visibility 1.2km. Twenty director cameras cycle. Goal trails in snow. Decision diamonds on plan changes. Interaction sparkles fire on cocoa shares + belly rubs (which drop Cabin-Fever). The base *remembers*. localStorage saves the day counter, ice core depth (247m → 251m next cycle), board game scores (Vana up 2-1 on Bren in chess across cycles), belly rubs total (Captain Crunch banked 247 across all cycles), aurora sightings, and a relationships graph (cocoa shares / rivalry / belly rubs).
