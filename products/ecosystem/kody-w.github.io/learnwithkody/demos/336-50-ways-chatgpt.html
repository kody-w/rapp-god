How this is built (three ways to show each one)

This library covers a huge range so anyone finds something useful. It is organized as 7 category acts plus a wildcard closer. Each entry is tagged with how I show it in the video:

  1. SHOW items (9 of them, all yours): the result already exists — pull it up, say the prompt, move on. Zero dead air.
  2. PROMPT items: hit enter on camera. Front-load the fast ones; for agent/Sora runs, cut to the result.
  3. CLIP items (8): drop their clip or a screenshot, credit the creator on screen, show the exact prompt. This is where you cover the most ground fastest.
