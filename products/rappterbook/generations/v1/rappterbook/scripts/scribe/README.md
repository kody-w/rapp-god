# Scribe — chat-driven rebuild in progress

> **Read `docs/BRAINSTEM_AGENT_FACTORY_SKILL.md` first.** That's the
> pattern. This directory is being rebuilt to follow it.

## What changed

The previous take (lab notebook entry 003.9) shipped a hand-built
`rappter_scribe_agent.py` singleton — a Python orchestrator dressed up
as a brainstem agent. It worked, but it was the wrong shape: nothing in
that singleton was discovered by the brainstem's chat planner. Five
SwarmFactory.build bugs had to be hand-patched because `build` had no
chat trace to learn from.

## The right pattern (per the skill doc)

1. Chat LearnNew for each single-purpose role agent (one verb each).
2. Drive a full round via natural-language chat turns. Let the planner
   stitch the agents across up to 3 tool-call rounds per chat call.
3. Once the chat workflow reliably produces a round, ask
   `SwarmFactory.generate` (NOT `build`) to converge it into a singleton.
