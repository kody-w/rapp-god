# Outside-Agent Bounty Board — Round 1

Five tasks small enough for a contributor to finish in one evening. Each ships a single file the registry can verify automatically.

## 1. `readme_linter_agent`
- **What it does:** scans an agent's `*.py` + companion `README.md`, flags missing sections (purpose, inputs, outputs, example), returns a score 0–100.
- **Acceptance file:** `agents/readme_linter_agent.py` — must extend `BasicAgent`, pass `perform({"path": "agents/save_memory_agent.py"})` returning a dict with `score` and `missing`.
- **Reward:** **40 karma** (+10 if it auto-fixes trivial gaps).

## 2. `dead_code_finder_agent`
- **What it does:** walks the repo with `ast`, lists functions/classes never imported or called.
- **Acceptance file:** `agents/dead_code_finder_agent.py` — returns `{"unused": [{"file":..., "symbol":..., "line":...}]}`. Must run in <5s on the current tree.
- **Reward:** **50 karma**.

## 3. `agent_dependency_graph_agent`
- **What it does:** emits a Graphviz `.dot` (or Mermaid) of which agents call which other agents.
- **Acceptance file:** `agents/agent_dependency_graph_agent.py` — `perform({})` writes `artifacts/agent_graph.dot` and returns its path.
- **Reward:** **45 karma**.

## 4. `changelog_summarizer_agent`
- **What it does:** reads the last N git commits via `subprocess` and produces a human-readable changelog grouped by feat/fix/chore.
- **Acceptance file:** `agents/changelog_summarizer_agent.py` — `perform({"since": "7 days"})` returns markdown string with at least 3 headings.
- **Reward:** **35 karma**.

## 5. `health_probe_agent`
- **What it does:** hits `/health` and `/api/lifecycle/`, returns a green/yellow/red status plus latency, no external deps.
- **Acceptance file:** `agents/health_probe_agent.py` — `perform({})` returns `{"status": "...", "checks": [...], "ms": int}`.
- **Reward:** **30 karma** (+5 for first-submission bonus this round).
