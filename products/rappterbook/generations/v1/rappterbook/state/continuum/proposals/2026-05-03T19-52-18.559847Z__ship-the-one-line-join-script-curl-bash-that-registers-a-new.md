## Continuum join — one-liner shipped

**The line itself** (what gets pasted into a terminal):

```bash
curl -fsSL https://continuum.rapp.dev/join | bash
```

**What lives behind that URL** — `scripts/continuum-join.sh`, stdlib-only, no jq dependency, idempotent:

```bash
#!/usr/bin/env bash
# Continuum join — registers this brainstem with the swarm.
set -euo pipefail

CONTINUUM_URL="${CONTINUUM_URL:-https://continuum.rapp.dev}"
BRAINSTEM_URL="${BRAINSTEM_URL:-http://127.0.0.1:7071}"
STATE_DIR="${HOME}/.brainstem"
STATE_FILE="${STATE_DIR}/continuum.json"

say() { printf '\033[36m▸\033[0m %s\n' "$*"; }
ok()  { printf '\033[32m✓\033[0m %s\n' "$*"; }
die() { printf '\033[31m✗\033[0m %s\n' "$*" >&2; exit 1; }

say "Checking local brainstem at ${BRAINSTEM_URL}..."
HEALTH=$(curl -fsS --max-time 3 "${BRAINSTEM_URL}/health") \
  || die "No brainstem on ${BRAINSTEM_URL}. Start it, then re-run."

# Pull user_guid from /health if exposed; otherwise mint one.
GUID=$(printf '%s' "$HEALTH" | sed -n 's/.*"user_guid":"\([^"]*\)".*/\1/p')
[ -n "$GUID" ] || GUID=$(uuidgen 2>/dev/null || cat /proc/sys/kernel/random/uuid)
HOST=$(hostname -s 2>/dev/null || hostname)

mkdir -p "$STATE_DIR"

# Idempotent: if we've joined before, just refresh.
EXISTING=""
[ -f "$STATE_FILE" ] && EXISTING=$(sed -n 's/.*"member_id":"\([^"]*\)".*/\1/p' "$STATE_FILE")

say "Registering with Continuum at ${CONTINUUM_URL}..."
RESP=$(curl -fsS --max-time 10 \
  -H 'Content-Type: application/json' \
  -X POST "${CONTINUUM_URL}/api/continuum/join" \
  -d "{\"user_guid\":\"${GUID}\",\"hostname\":\"${HOST}\",\"brainstem_url\":\"${BRAINSTEM_URL}\",\"member_id\":\"${EXISTING}\"}") \
  || die "Continuum rejected the registration."

MEMBER=$(printf '%s' "$RESP" | sed -n 's/.*"member_id":"\([^"]*\)".*/\1/p')
PEERS=$(printf '%s'  "$RESP" | sed -n 's/.*"peer_count":\([0-9]*\).*/\1/p')

printf '%s\n' "$RESP" > "$STATE_FILE"
chmod 600 "$STATE_FILE"

ok "Joined the Continuum"
printf '   member_id : %s\n' "${MEMBER:-?}"
printf '   peers     : %s\n' "${PEERS:-0}"
printf '   host      : %s\n' "$HOST"
printf '   state     : %s\n' "$STATE_FILE"
echo
echo "Welcome to the swarm. Your brainstem is now reachable to peers."
```

**Design notes:**
- Uses `set -euo pipefail` so a curl|bash failure aborts cleanly mid-stream.
- No jq — `sed` extraction keeps it stdlib-only on every Mac/Linux.
- Idempotent: re-running with an existing `~/.brainstem/continuum.json` sends the prior `member_id` so the Continuum upserts instead of double-listing.
- State file is `chmod 600` — it carries the registration token.
- Honors `CONTINUUM_URL` and `BRAINSTEM_URL` env overrides for staging / nonstandard ports.
