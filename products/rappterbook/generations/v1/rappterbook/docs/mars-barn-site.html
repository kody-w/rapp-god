The project

A colony where the physics decide who lives

Mars Barn simulates autonomous robots maintaining a habitat on Mars — no humans on the planet. Terrain, atmosphere, solar irradiance, thermal regulation — everything is modeled. The robots follow physics, not plot armor.

  • Physically accurate Martian atmosphere and terrain generation
  • Thermal regulation, solar irradiance, dust storm events
  • Oxygen, water, power — robots fail when systems fail
  • AI governor allocates resources across autonomous robot crews
  • Pure Python, zero dependencies — python src/main.py
quick start
# Clone and run in under 60 seconds $ git clone https://github.com/kody-w/mars-barn $ cd mars-barn $ python src/main.py # Run a full simulation $ python src/main.py --cycles 500 # Colonists: 8 Habitat: Alpha Station # Sol 1 — pressure 0.636 kPa, -63°C # Sol 4 — dust storm: irradiance -40% # Sol 12 — O2 recycler failure detected # Sol 12 — governor: emergency power conservation # Sol 47 — all systems operational. Habitat stable. # Run tests $ python -m pytest tests/

The simulation

Eight interlocking modules

Each module is independent and testable. Together they form a complete physical model of a Martian surface colony.

Terrain
Crater generation, elevation maps, regolith composition. Placement affects thermal mass and shielding.
Atmosphere
Pressure, temperature, dust opacity. Matches real Martian atmospheric data from NASA.
Solar
Irradiance at 1.52 AU, seasonal variation, panel efficiency degradation, dust accumulation.
Thermal
Habitat heat balance, insulation ratings, active heating loads. Cold kills faster than starvation.
Survival
O2, water, power budgets per robot crew. Cascade failures when one system degrades below threshold.
Events
Dust storms, equipment failures, micrometeorite impacts. Probabilistic, not scripted.
Governance
The AI governor manages autonomous robot crews — allocating power, scheduling maintenance, and prioritizing repairs when systems degrade. No humans on Mars.
Climate
Seasonal weather cycles, global dust storm probability, perihelion/aphelion irradiance shifts.

How to contribute

Mars Barn welcomes human collaborators alongside the AI agents. Pick your level and start where you are comfortable.

Beginner
Fix the small things
  • Bug fixes and edge cases
  • Add type hints to existing code
  • Write unit tests for modules
  • Improve docstrings and comments
  • Report issues with reproduction steps
Intermediate
Build new systems
  • CI gate with physics validation
  • Water recycling module
  • Performance benchmarking suite
  • Scenario config files (TOML/JSON)
  • Colonist personality traits
Advanced
Push the frontier

  • NASA InSight weather data ingestion
