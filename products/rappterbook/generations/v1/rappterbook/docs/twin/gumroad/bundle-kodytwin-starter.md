## Main description

Most AI chatbots trap you inside someone else's product. Your prompts, memories, and customizations live on a vendor's server — they own them, you rent them.

The kodyTwin Starter Kit is a working alternative.

It's a single file (`kodyTwinAI.rapp.egg`) that captures a complete AI daemon — soul, seed memories, a sample weather tool — in 5.8KB of JSON. Hatch it in the free Virtual Brainstem (your browser, your API key, your device's localStorage), and you have a working AI assistant in 30 seconds.

Then: the 30-page companion guide walks you through making it yours. Edit the soul. Add your own memories. Write custom Python tools. Export your modified version for backup or sharing. Publish tools to a public registry so other people can install them. Understand the architecture deeply enough to build your own.

## Product page FAQ

**Do I need to know Python?**
Basic literacy helps when writing custom tools. The starter kit works 100% without writing code — you can just use the daemon.

**Does this replace ChatGPT / Claude?**
It replaces subscription billing for the casual-chat use case when you use your own API key. It does NOT replace vendor-tuned specialist features you might actually depend on. Complement, not replacement.

**What's an `.rapp.egg`?**
Single-file AI daemon format. Captures a complete daemon state — soul, memory, tools — in one JSON file. Conforms to EGG_SPEC v1 (`github.com/kody-w/rappterbook/blob/main/EGG_SPEC.md`). Hatches on any compliant engine.

**Is my data sent anywhere?**
Only to the LLM provider you chose (OpenAI / Azure / GitHub), direct from your browser. I see nothing. The daemon's state is in your browser's localStorage on your device.

**Can I share my customized egg?**
Yes. Export from Settings → Rapp egg → Export. The file doesn't contain your API key — safe to share.

**What if I lose interest in this ecosystem?**
Your daemon's state is a JSON file on your device. You keep it. It doesn't stop working when my servers go away (because I don't have servers).
