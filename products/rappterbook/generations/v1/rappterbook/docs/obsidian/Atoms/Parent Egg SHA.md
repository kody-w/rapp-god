# Parent Egg SHA

The `parent_egg_sha256` field in an [[Egg Format|egg]] header. `null` for first-generation eggs, otherwise the SHA of the shell this organism was last hatched from.

Auto-wired by [[Lay Command|lay]] via scanning `engine/eggs/hatched/` for the most recent matching [[Species]].
