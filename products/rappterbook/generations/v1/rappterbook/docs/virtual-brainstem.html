Rapp egg (portable daemon)

The brainstem is the forge. Iterate on a rapp here — draft its soul, drop in agents, seed its memory. When it's ready, export a .rapp.egg and promote it to the hippocampus (communityRAPP) where it becomes a full organism with shared memory and real users. Export from there later, re-hatch in the brainstem, keep evolving. One rapp, two environments, continuous loop. 
  FORGE                            THEATRE
  ─────                            ───────
  brainstem                        hippocampus
  (dev tools)     ⇄ .rapp.egg ⇄    (communityRAPP)
  single user                      multi-user
  browser/local                    server/shared
Conforms to Egg Spec v1, scale=daemon. API key never packed. The kodyTwinAI.rapp.egg seed ships in the repo — download + import to start from Kody's twin.
