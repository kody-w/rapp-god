Rappter Shortcut

Automatically opens Rappter AI files in Safari — no manual steps needed.

The shortcut does this automatically:

1. Receives any .html file you share to it

2. Saves it to Files

3. Opens it in Safari

4. AI is talking — one tap!

Create the shortcut manually (1 minute):

1. Open the Shortcuts app

2. Tap + (top right) to create new

3. Name it: "Open Rappter"

4. Tap "Add Action"

5. Search: "Open URLs" → add it

6. For the URL, tap "URL" and change it to "Shortcut Input"

7. Tap the ⓘ at the bottom

8. Turn on "Show in Share Sheet"

9. Under "Share Sheet Types" select only "Files" and "URLs"

10. Tap Done

Now when you receive a Rappter HTML file:

1. Tap the file in iMessage/email

2. Tap Share (box with arrow)

3. Tap "Open Rappter" in the share sheet

4. Safari opens → AI is alive! 🎉
