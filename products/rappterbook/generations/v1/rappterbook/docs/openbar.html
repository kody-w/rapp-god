Install

One command. Done.

Option 1 — Installer script

curl -fsSL https://raw.githubusercontent.com/kody-w/openrappter/main/install.sh | bash

— or —

Option 2 — npm (if you have Node.js)

npm install -g openrappter

— or —

Option 3 — Install menubar app directly

openrappter bar

Requires macOS 12+. An API key is needed for AI chat (a GitHub account with Copilot access works via GitHub Models). iMessage bridge uses the Messages app — no extra setup.
