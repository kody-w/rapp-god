🧠 Dream Mode — Your AI Cleans Its Own Brain

Every night at 3am, the Dream agent wakes up, reviews all stored memories, and consolidates them. It finds duplicates via token similarity, prunes stale entries that were never accessed, and logs what it cleaned.

On the first run, it found that "vitest is the test framework" had been stored 90+ times. Dream mode consolidated 123 memories down to 21 unique ones — 83% bloat removed in one command.

# Full dream cycle openrappter --exec Dream "" # Dry run — see what would be cleaned openrappter --exec Dream "audit" # Schedule nightly at 3am (already done during onboard) { "schedule": "0 3 * * *", "agentId": "Dream" }

Like biological sleep for AI. The system gets cleaner the longer it runs.

👻 Soul Templates — 10 Prebuilt Personalities

Soul templates let you summon specialized versions of your rappter. Each one defines which agents are included, a system prompt personality, and an emoji identity.

💻 Coder — Writes code, runs tests, ships PRs
🔬 Researcher — Searches, reads, synthesizes
🛠 DevOps — Monitors, heals, deploys, alerts
🎙 Narrator — Voice-first, speaks everything via TTS
🔮 Oracle — The AI that improves the AI
💬 Companion — Warm, remembers everything, chats anywhere

Plus: assistant, reviewer, analyst, and scheduler. Load any with one API call.

📬 30-Day Onboarding Tips

After setup, you get one native macOS notification per day at 9am. Each teaches a single feature with a command to try. Click the notification to open the app.

Sample schedule:

  • Day 1: "I'm your AI agent running in the background. Click to chat."
  • Day 5: "I have 22 specialized agents. See them all: --list-agents"
  • Day 11: "Dream Mode cleans my brain every night. Run it: --exec Dream audit"
  • Day 15: "Create your own agents. Describe one in English."
  • Day 22: "Watchmaker evaluates quality and promotes winners. Darwinian evolution."
  • Day 30: "You've graduated! 🎓"

🧙 Visual Onboarding — No Terminal Required

Non-technical users can now set up openrappter entirely from the menu bar app. Install the DMG, click the 🦖, and a SwiftUI wizard walks you through:

  1. Welcome — meet your dino
  2. GitHub auth — device code flow (opens browser)
  3. Telegram — optional bot connection
  4. Auto-start — daemon launches, launchd installs, cron activates
  5. Done — transitions to chat, first tip fires

Two paths to the same destination. Terminal for developers, menu bar for everyone else.

⚡ Everything Else

🔄 Self-Updating — New Update agent checks GitHub for releases, pulls, and rebuilds. Plus a new automated release workflow (tag → test → changelog → GitHub Release → npm publish).
⏱ Cron Actually Works Now — The CronService was never wired into the gateway startup. Fixed. Your cron jobs (sweet-nothings, morning-brief, dream-mode, daily-tip) now actually fire.
🌅 MorningBrief Agent — Chains Web (weather) → Calendar → Memory (priorities) → TTS (speak). Runs at 8am daily.
🔍 AgentRegistry Discovery — User-generated agents (from LearnNew) are now auto-discovered and available in --list-agents and --exec.
💾 Soul Persistence — Save, load, and delete soul configs to ~/.openrappter/souls/*.json. Souls survive daemon restarts.
🐛 NetworkDiscovery Crash Fix — Fixed a double-resume race condition in CheckedContinuation that was crashing the menu bar app.
