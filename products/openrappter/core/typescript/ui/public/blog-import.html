rappter.com (GoDaddy)

Step 1: Log in to WordPress

1

Open your WordPress admin in a new tab:

https://kodyw.com/wp-admin/

Log in with your username and password. Keep this tab open.

Step 2: Drag this bookmarklet to your bookmarks bar

Or: copy the code below and create a bookmark manually with this as the URL.

Step 3: Click the bookmarklet on your WordPress admin page

3

While on kodyw.com/wp-admin/, click the "Import Posts" bookmarklet. It will use the WordPress REST API to create all 4 posts as drafts.

Posts are created as drafts — review and publish when ready.

rappter.com — GoDaddy Website Builder

GoDaddy's builder doesn't have a REST API. Here's how to add blog posts:

1

Log in to GoDaddy:

https://www.godaddy.com/websites/website-builder
2

Open your rappter.com site editor → Add Section → Blog

If you haven't enabled the blog feature yet, it'll prompt you to create one.

3

For each post below, click Copy HTML, then create a new blog post in GoDaddy and paste the content.
