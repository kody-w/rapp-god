# rapp-vneighborhood

**Front doors & neighborhoods** — the front-door template (a public repo *is* the front door).

- Canonical: https://github.com/kody-w/rapp-vneighborhood
- Schema: `rapp-vneighborhood/1.0`
- Default branch: `main`

## What it is

The **template for planting a neighborhood front door**. The key idea: a public GitHub repo *is* the front door — no separate server. The `v` means swarm-capable. The relay behind it is interchangeable (local ≡ kited ≡ cloud), so the same neighborhood can be served from a laptop, a kited twin, or a cloud relay without changing the door.

Fork it, fill in your neighborhood's identity (rappid, soul, members, card), and you have a planted gate with egg/import/fork portability built in.

## What it provides

- The canonical neighborhood front-door scaffold.
- Interchangeable-relay serving (local / kited / cloud).
- Egg-import + fork portability out of the box.

The archetype neighborhood is [rapp-commons](rapp-commons.md) (the lowest possible floor: scan, hatch, post a hello). Example neighborhoods built from this template include `vneighborhood-design-studio` and `vneighborhood-research-lab`.
