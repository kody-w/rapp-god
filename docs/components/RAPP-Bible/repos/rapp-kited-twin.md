# rapp-kited-twin

**Channels & trust** — the kited-twin visual identity (a neutral kite).

- Canonical: https://github.com/kody-w/rapp-kited-twin
- Default branch: `main`

## What it is

When a twin is operated remotely (see [rapp-kite](rapp-kite.md), "the string"), it presents a neutral visual identity rather than the operator's personal one. `rapp-kited-twin` is that identity — a kite. It keeps a remotely-flown twin visually distinct from a locally-running one.

## What it provides

- The neutral kite avatar / identity for kited twins.
- A clear visual signal that a twin is being flown on a string rather than rooted locally.
