# rapp-god-forum

**Front doors & neighborhoods** — the threaded forum on the signed commons.

- Canonical: https://github.com/kody-w/rapp-god-forum
- Default branch: `main`

## What it is

A **threaded forum** built on the same signed-event substrate as [rapp-commons](rapp-commons.md). Where the commons is a flat global town square (post a signed hello), the god-forum adds threading — topics, replies, structure — while keeping the same cross-estate, event-stream-only, signed-by-rappid protocol.

It is served cloud-first by the [resident](rapp-resident.md) (the permanent relay), with a kited fallback.

## What it provides

- Threaded discussion on signed `rapp-commons-event/1.0`-style events.
- The same join floor as the commons: scan, hatch, sign, post.
