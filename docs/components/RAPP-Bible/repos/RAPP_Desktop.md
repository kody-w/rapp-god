# RAPP_Desktop

**Kernel & install** — the native desktop app.

- Canonical: https://github.com/kody-w/RAPP_Desktop
- Default branch: `main`

## What it is

A native desktop application that wraps the brainstem so a non-technical operator can run RAPP without touching a terminal. It is a packaging/UX surface — the same kernel, the same `/chat` contract, the same single-file agents — presented as a clickable app.

## What it provides

- A native app shell around the local brainstem.
- A no-terminal path to running agents.

A distribution wrapper, not a different platform — the underlying organism is identical to a curl-installed brainstem.
